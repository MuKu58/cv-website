# 💼 Interactive CV Website – Muhammed Kurkcu

A modern and interactive CV website built with HTML, CSS, and JavaScript.  
This project transforms a traditional CV into a responsive web experience, showcasing both my professional background and my growing skills as a frontend developer.

---

## 🚀 About the Project

This CV website is designed to present my profile in a more dynamic and engaging way than a standard PDF.

It combines:
- A structured CV layout
- Interactive UI elements
- Responsive design for all devices
- A printable version for traditional use

---

## 🧠 My Background

I am currently transitioning from healthcare to software development.  
Through this project, I focus on building strong frontend fundamentals and applying them in a real-world use case.

---

## ✨ Features

- 📱 Responsive layout (desktop & mobile)
- 📂 Sidebar CV structure (like a real resume)
- 🍔 Mobile navigation with hamburger menu
- 🔽 Dropdown menu for additional sections
- ⚡ Smooth scrolling navigation
- 🖼️ Profile photo upload with preview
- 💾 LocalStorage support (saved profile image)
- 🖨️ Print-friendly CV layout (for PDF export)
- 📅 Auto-updating “last updated” date in footer

---

## 🛠️ Technologies Used

- HTML5 (semantic structure)
- CSS3 (layout, styling, glassmorphism, responsive design)
- JavaScript (interactivity, DOM manipulation, localStorage)
